# job-e-diary

This is a Jaxxon Concrete Pumps project.

To get setup, use the following git commands:
1. mkdir repo
2. cd repo
3. git init
4. `git remote add origin https://github.com/hyperdrive3/job-e-diary.git`
5. `git fetch origin`
6. `git checkout master`
  
In VS Code you can create a new folder, go to the Source Control panel and Initialize Repository, then continue the commands from git remote add.
